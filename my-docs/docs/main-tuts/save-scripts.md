# Saving Scripts on iOS
When you delete Delta (to update it), the scripts you saved will be deleted! However, there is an easy way to save them.

## Instructions
1. Open the Files app on your device.
2. Go to Browse > On My (Device) > Delta > Delta.
3. Hold the Scripts folder, then click Move.
4. Click Browse. Save the folder in a place you will remember (e.g., Downloads).
5. After you reinstall Delta, find the Scripts folder you saved. Hold it, then click Move.
6. Go to Browse > On My (Device) > Delta > Delta.
7. Click Move. If prompted, click "Replace" (so the empty Scripts folder is replaced by the one you saved earlier).
